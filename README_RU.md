# LessonRecorder

## Что делает программа

1. **Записывает** звук с микрофона и/или системного аудио (захватывает онлайн-уроки)
2. **Распознаёт** речь офлайн через Whisper AI — без интернета
3. **Генерирует** структурированный конспект через бесплатный облачный AI (Groq, Gemini, OpenRouter)
4. **Сохраняет** всё в историю уроков с поиском и экспортом
5. **Обновляется** автоматически — сообщает о новых версиях при запуске

## Установка

1. Открой [Releases](../../releases) и скачай `LessonRecorder_X.X.X_setup.exe`
2. Запусти установщик — права администратора не нужны
3. Открой **LessonRecorder** из меню Пуск или с рабочего стола

## Первоначальная настройка

### Транскрипция (работает офлайн)

Открой **Настройки → Транскрипция** и выбери модель Whisper:

| Модель | Скорость | Качество | Когда использовать |
|--------|----------|----------|--------------------|
| `tiny` | ⚡ Очень быстро | Хорошее | Быстрые заметки, чёткое аудио |
| `base` | Быстро | Лучше | Рекомендуется по умолчанию |
| `small` | Средне | Отлично | Важные лекции |
| `medium` | Медленно | Превосходно | Максимальное качество |

Модель скачивается автоматически при первом использовании и кешируется локально.

### AI конспект (бесплатно, нужен интернет)

Открой **Настройки → AI Провайдер** и выбери:

| Провайдер | Бесплатный лимит | Регистрация |
|-----------|-----------------|-------------|
| **Groq** | 14 400 запросов/день | [console.groq.com](https://console.groq.com) |
| **Google Gemini** | 250 запросов/день | [aistudio.google.com](https://aistudio.google.com) |
| **OpenRouter** | 50 запр/день (бесплатные модели) | [openrouter.ai](https://openrouter.ai) |
| **Свой URL** | Любой OpenAI-совместимый API | — |

Карта не нужна ни для одного из них. Вставь ключ → нажми **Проверить соединение** → Сохранить.

## Как пользоваться

1. Открой вкладку **Запись**
2. Выбери источник: Микрофон / Системный звук / Оба
3. Нажми **Начать запись**
4. Когда закончишь, нажми **Стоп** — транскрипция запустится автоматически
5. После транскрипции нажми **Создать конспект** для AI-резюме
6. Все записи хранятся во вкладке **История**

## Автообновление

Когда выходит новая версия, приложение показывает уведомление при запуске:

> **«Доступно обновление v1.X.X»**  
> Скачать и установить?

Нажми **Скачать** — установщик запускается в фоне и перезапускает приложение. Никаких ручных действий не нужно.

## Советы

- Используй **«Оба источника»** чтобы записать и свой голос, и звук онлайн-урока
- Первая транскрипция занимает ~30 сек (загрузка модели) — последующие мгновенные
- Одно слово и короткие фразы распознаются нормально — фильтр тишины отключён
- Если Groq недоступен в твоём регионе — переключись на Google Gemini, он работает везде
